# ✨ Luciaaaa Beauty E-commerce

[![Luxury](https://img.shields.io/badge/Aesthetic-Luxury-gold.svg)](https://github.com/0xUnusual/Luciaaaa-Beauty)
[![Status](https://img.shields.io/badge/Status-Beta-brightgreen.svg)](https://github.com/0xUnusual/Luciaaaa-Beauty)

**Luciaaaa Beauty** es una experiencia de e-commerce premium, diseñada para productos de Skincare con una estética minimalista, femenina y altamente interactiva. Inspirada en los estándares de diseño de alta gama como *Glossier*, *Rare Beauty* y *Apple*.

---

## 💎 Características Principales (UX/UI Premium)

- **Hero Cinemático con Scroll-Trigger**: Un inicio impactante donde el contenido aparece y desaparece sutilmente mediante efectos de "Pin & Scrub" de GSAP.
- **Glassmorphism Header**: Un Navbar y Promo-Bar translúcidos que aparecen con un efecto de desenfoque al hacer scroll.
- **Búsqueda Reactiva (Live Search)**: Búsqueda instantánea de productos con micro-tarjetas de resultados y botón de añadido directo.
- **Sistema de Notificaciones Toast**: Alertas flotantes elegantes (estilo iOS) que confirman la adición de productos sin interrumpir la navegación.
- **Checkout Optimizado**: Un flujo de pago fluido con validaciones en tiempo real para tarjetas y formularios, simulando una pasarela Stripe.
- **Mobile-First & Ultra Responsive**: Cada pixel ha sido pulido para una visualización perfecta en cualquier dispositivo.

---

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5 Semántico, CSS3 Vanilla (Custom Properties), JavaScript Vanilla (ES6+).
- **Animaciones**: 
  - [GSAP](https://greensock.com/gsap/) (ScrollTrigger para el Hero).
  - [AOS](https://michalsnik.github.io/aos/) (Scroll Reveal en secciones).
- **Estado**: Centralized State Management (Store pattern) para el carrito y búsqueda.
- **Estética**: Google Fonts (Inter, Playfair Display), Flexbox/Grid Layouts, Glassmorphism.

## 🚀 Instalación y Uso

Para ejecutar este proyecto de forma local:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/0xUnusual/Luciaaaa-Beauty.git
   ```
2. **Entra al directorio:**
   ```bash
   cd "Beauty LOP"
   ```
3. **Abre el proyecto:**
   - Simplemente abre `index.html` en tu navegador o usa una extensión de tipo **Live Server** para la mejor experiencia.
